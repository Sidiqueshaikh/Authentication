Firebase Login & Register System

A simple and responsive Login & Registration system built using HTML, CSS, JavaScript, and Firebase.
This project allows users to:

Register with email & password
Login securely using Firebase Authentication
Store user details in Firebase Firestore
Auto-expire user session after 1 hour
Switch between Login and Register forms dynamically

Features:
User Registration
User Login
Firebase Authentication
Firestore Database Integration
Session Management
Auto Logout after 1 Hour

Technologies Used:
HTML5
CSS3
JavaScript
Firebase Authentication
Firebase Firestore
